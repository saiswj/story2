# 《都分手了，她表妹怎么追来了》整书走向推演

> 按450章完结版推演。主线坚持番茄男频口径：反舔、听劝、前任后悔、表妹倒追、公开打脸、创业起飞。每5章有钩子，每10章有外部爽点，每30章左右有公开爆点。

> 108万轻补版方向：从卷01后半开始补公开场面和外部压力，不改主线、不大扩章数。新增人物以功能型为主：卷01竞赛/生日宴见证者，卷02社团与用户代表，卷03强化孙启明承担竞品产品/补贴压力且直播主持匿名化，卷04数据核验员，卷05外校合作方与挖人线，卷06大厂法务/产品压力。详见 `108万轻补与新增人物方案.md`。

> 落地状态：正文轻补已完成，保持450章不变；正文源文件非空白字符约105.10万，根目录合订版含标题非空白字符约105.97万。主补内容集中在公开压迫、用户可见处理、合作暂停权、权限矩阵、竞品短暂胜利、大厂条款和团队最终表决，不改变陆远、林夏、许晴三条主线。

## 全书主问题

陆远前世把人生价值押在许晴一句认可上，重生后绑定听劝系统。全书真正要回答的问题不是“他能不能变强”，而是：

他能不能从“听别人劝才变强”，走到“听完真话后，自己负责选择”。

林夏的价值不是替代许晴成为新的奖励，而是让陆远看见：真正亲近的人不是享受他的付出，而是敢说难听真话、也愿意和他一起承担结果。

## 卷一：1-30章，重生后，我不舔了

核心走向：

- 陆远重生到大二开学，距离许晴分手三天，手里还拎着给前女友的早餐。
- 第一章用前世回忆让林夏亮相：“你就是我姐那个前男友？”
- 陆远第一次不送早餐，听劝系统激活。
- 许晴从“他又闹脾气”变成“不对，他真不来了”。
- 林夏第8章正叙登场，公开把书名钩子打出来。
- 林夏从替表姐审判陆远，到因为他的执行力和结果第一次动摇。
- 第30章生日宴公开爆点：周正阳设局，陆远拿项目晋级反打，林夏护场，许晴第一次低头。

卷末状态：

- 陆远建立边界。
- 许晴第一次失去掌控感。
- 林夏从旁观审判者变成偏心前夜。
- 周正阳从嘴炮对手升级为明面反派。

## 卷二：31-80章，表妹开始偏心了

核心走向：

- 生日宴余波发酵，论坛开始讨论“前任、表妹、陆远翻身”。
- 陆远不沉迷修罗场，直接把竞赛Demo拆成校园产品，第35章前完成冷启动。
- 林夏从嘴硬“我只是看不惯造谣”，变成行动上不断站陆远。
- 许晴看到林夏和陆远越来越默契，前任优越感崩塌。
- 周正阳抢展示位、抢资源，但陆远用真实用户数据反打。
- 第80章，许晴求复合被拒，林夏主动宣战：“我不是替你看他，我是自己想靠近他。”

卷末状态：

- 陆远完成从竞赛选手到产品发起人的过渡。
- 林夏正式从“表姐阵营”转为“陆远阵营”。
- 许晴进入强后悔线。
- 前任后悔和表妹倒追同时明牌。

## 卷三：81-150章，前女友求复合

核心走向：

- 许晴求复合失败后开始反思，但她的反思来得越晚，读者越爽。
- 林夏主动约饭、改方案、拉合作，倒追动作明显增加。
- 校园产品冲上榜单，用户、付费、口碑全部外显。
- 周正阳加入竞品项目，开始用补贴和抄袭正面围剿。
- 林夏通过直播演示竞品差距，独立立功。
- 周正阳攻击林夏“抢表姐前男友”，逼情感线公开。
- 第150章，林夏当众告白，陆远没有立刻答应，但第一次明确护她。

卷末状态：

- 林夏公开追，陆远默认她靠近但保留边界。
- 许晴彻底明白自己回不去了。
- 周正阳第一次大败，但还没有彻底失去资源。
- 秦川注意到陆远团队，事业线进入资本视野。

## 卷四：151-230章，表妹公开倒追

核心走向：

- 林夏已经公开告白，陆远开始动心，关系进入半公开暧昧。
- 两人继续以项目靠近，避免空甜。
- 路演前夜，林夏作为有效建议源，帮陆远把路演稿打磨到S级。
- 周正阳联合竞品攻击两人关系，试图把事业成绩抹成情感炒作。
- 陆远用留存率、付费率、用户增长打脸。
- 唐明远公开认可团队结果，秦川当场递出名片并约看原始后台。
- 舆论继续攻击林夏“抢表姐前男友”，许晴第一次替林夏说话。
- 第230章，陆远公开确认林夏是合伙人，不是附属；林夏获得“合伙人+准女友”双身份。

卷末状态：

- 感情线从“追不追”变成“能不能并肩”。
- 许晴从破防后悔转向体面退场前奏。
- 事业线从校内项目转向真实商业竞争。
- 秦川入局，带来机会和压力。

## 卷五：231-350章，创业起飞

核心走向：

- 产品从本校扩散到周边学校，用户破十万，付费爆单。
- 陆远第一次被真实用户骂到破防，但仍然照改，体现“听劝”升级为“听市场真话”。
- 秦川提出投资条件，条件诱人但要求快速扩张。
- 团队出现方向分歧：马成海想快，韩子昂想稳，林夏想保用户口碑；冲突外化为一万八首屏广告取舍、会员试点公开追问、退款和适用范围误会。
- 顾明、群管理员和南湖对接人把分层、低价、稳定三类矛盾推到外部，逼团队在公开场面里给处理路径。
- 周正阳加入竞品公司，靠低价、豪华团队、挖人、偷运营方案围剿。
- 周正阳靠补贴短暂赢到外部结果：城西理工和南湖师院两个入口被云习社抢走，团队承认真实损失，不再把失败写成漂亮长期主义。
- 林夏和韩子昂把反击从内部复盘改成外部验收：城西老用户现场复测、南湖师院十个测试账号、灰度现场连线。
- 周正阳改打团队人心，赵宇追问岗位位置，陈浩然被竞品头衔晃动；陆远不靠情绪挽留，而是补岗位边界和交接规则。
- 云习社补贴规则翻车后，学伴不踩对手，先把付款规则前置；林夏独立拿下外语学院三门课试点，并获得合作边界暂停权。
- 秦川把最后考察放到试点现场，看学伴如何拒绝不适用用户付费；周正阳后续试图甩锅陈浩然，反被时间线和内部流程反噬。
- 卷05后段以全员权限重构、四校名单公开和“能买/待核验/暂不收费”三类边界，兑现估值继续上调。
- 陆远不再依赖系统给标准答案，而是做用户分层和产品重构。
- 新版本上线后留存、转化反超竞品。
- 第350章，估值暴涨，许晴亲戚反过来求陆远，林夏用一句“现在也挺好，只是身份变了”完成旧人反转。

卷末状态：

- 陆远成为真正的团队负责人。
- 林夏证明自己不是女主挂件，而是能独立拿合作、看资本坑的合伙人。
- 周正阳从校园反派升级为商业失败者；卷中段先短暂得手，再被留存、退款和反馈响应公开反杀。
- 许晴彻底从“后悔的人”变成“见证错过的人”。

## 卷六：351-450章，真正的选择

核心走向：

- 大厂提出收购，报价足以让团队财富自由，但要求放弃部分产品方向和控制权。
- 团队成员意见分裂，系统不再直接发奖励，只提醒陆远：建议越多，越要知道自己要什么。
- 周正阳试图借资本和舆论逼陆远低头。
- 团队通过合同、数据和公开证据反杀，周正阳资源、名声、项目全线崩盘。
- 许晴轻量体面退场，承认自己怀念的是过去那个围着她转的人，不是现在的陆远。
- 系统发布最后任务：听取所有关键建议后，做出自己的选择。
- 陆远接受投资，但拒绝控制权转让。
- 第450章，林夏正式成为公开女友和公司合伙人；系统沉默，陆远仍继续听用户、团队、林夏的真话。

终局状态：

- 陆远完成主题闭环：听劝不是没主见，而是听完之后敢负责。
- 林夏完成关系正名：她不是前女友的表妹附属，而是陆远选择的爱人和合伙人。
- 许晴完成体面退场：承认错过，不再争夺。
- 周正阳完成反派终局：靠包装和资源的人，最终败给真实产品和公开证据。
