# 第279章：林夏谈下第二所学校

- 所属卷：卷五：学伴杀出校园
- 故事阶段：外校入口保卫
- 本章作用：林夏单独谈下第二所学校有限试点，给学伴保住外校扩张入口
- 出场人物：林夏、魏敏、陆远、马成海、韩子昂、刘鹏、李小薇

## 事件推进

1. 魏敏追问谁负责、怎么停、何时复核，林夏把全量开放改成有限试点。
2. 项目教室里，陆远让团队先接住老师三问，韩子昂核权限，刘鹏补用户原话，马成海删掉夸大标题。
3. 魏敏给出南湖师院、城西理工、南江财大的分层试点条件，林夏把入口改成试点观察入口。
4. 公告发出后，老用户、外校学生和魏敏连续追问，团队用处理队列、投诉入口和十个测试账号接住第一晚压力。

## 结果落点

第二所学校没有撤入口，学伴拿到有限试点和十个测试账号，老用户确认不会被挤到后面，但团队必须今晚跑通第一条错资料处理流程。

## 连续压力

外校入口保住后，投资谈判先让位于真实试点验证。
