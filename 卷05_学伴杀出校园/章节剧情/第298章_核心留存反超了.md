# 第298章：核心留存反超了

- 所属卷：卷五：学伴杀出校园
- 故事阶段：团队内耗与产品重构
- 本章作用：落地留存反超后的情绪反应，同时让马成海学会赢后先补未完成项
- 出场人物：陆远、林夏、马成海、韩子昂、刘鹏、秦川、周正阳、孙启明、南湖群主、魏敏、张博

## 事件推进

1. 马成海差点翻椅子想立刻发图，林夏按住他先看退订理由。
2. 云习社只发新增，评论区追问留存；学伴发留存曲线，同时附上未完成适配课程列表。
3. 顾明评论提醒城北待核验课仍要下午六点落地，陆远把这条置顶，压住只晒成绩的冲动。
4. 用户评论“这就比只发新增强”，周正阳被迫沉默，秦川要求下午看未完成列表如何减少。
5. 南湖群主要求提前全开入口，马成海没有再秒回“快了”，而是只开放已核验课程。

## 结果落点

留存曲线交叉、学伴公开留存图和未完成列表、顾明置顶评论、云习社新增海报下的追问、南湖群主同意只开已核验课程。

## 连续压力

马成海开始承认自己太急，第299章进入团队内部成长和运营打法调整。
