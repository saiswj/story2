# 第330章：周正阳开始甩锅

- 所属卷：卷五：学伴杀出校园
- 故事阶段：竞品围剿
- 本章作用：压缩周正阳嘴炮，改成被补贴失控和内部流程反噬
- 出场人物：周正阳、孙启明、陆远、林夏、马成海、韩子昂、刘鹏、陈浩然、许晴、秦川、赵宇

## 事件推进

1. 周正阳把陈浩然聊天记录包装成外校运营失误，陈浩然在学伴项目教室当场发白。
2. 陆远要求陈浩然导出完整聊天记录，不替他遮动摇，也不让他背不属于他的锅。
3. 学伴发布时间线说明，证明补贴规则和客服话术早于陈浩然接触云习社。
4. 孙启明逼周正阳写清补贴规则制定流程，云习社内部开始反噬。

## 结果落点

云习社甩锅截图、陈浩然完整聊天记录、学伴时间线说明、孙启明会议桌上的空白流程。

## 连续压力

陈浩然看清空头衔背后的锅，第331章转入他后悔却无法回到核心岗位的后果。
