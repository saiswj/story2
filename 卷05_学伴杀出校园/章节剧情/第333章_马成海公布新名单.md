# 第333章：马成海公布新名单

- 所属卷：卷五：学伴杀出校园
- 故事阶段：竞品围剿
- 本章作用：把新学校名单变成社群公开大场面，落地扩张、拒绝乱卖和估值继续上调
- 出场人物：陆远、林夏、马成海、韩子昂、刘鹏、魏敏、赵宇、秦川助理、各校用户

## 事件推进

1. 新学校名单一发出，四所学校用户同时追问，团队按学校分窗口接住。
2. 马成海在工业大学群里解释为什么只查询不收费，被用户追问后改成直接说明“现在不能买”，赵宇负责把外校窗口分清，林夏在外语学院当场拒绝超范围付费。
3. 陆远拒绝南江财大未核验用户主动付费，论坛出现“怪，但比乱卖强”的自发评价。
4. 秦川助理确认下一轮报价可继续上调，但要求每所学校都写清开放、待核验、暂不收费三类边界。

## 结果落点

四校名单公告、四个社群窗口、魏敏确认边界、赵宇执行分工、论坛“怪，但比乱卖强”帖子、秦川下一轮报价条件。

## 连续压力

新学校名单稳住扩张口碑，孙启明的低价打法被进一步反衬，学伴进入投资报价继续上调阶段。
