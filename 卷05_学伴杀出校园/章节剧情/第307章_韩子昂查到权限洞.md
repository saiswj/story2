# 第307章：韩子昂查到权限洞

- 所属卷：卷五：学伴杀出校园
- 故事阶段：团队内耗与产品重构
- 本章作用：韩子昂发现临时协作账号仍保留权限，团队第一次把“信任”和“可追责边界”拆开
- 出场人物：韩子昂、陆远、林夏、马成海、陈浩然、魏敏、赵宇

## 事件推进

1. 韩子昂发现外校活动页协作时开的临时账号仍有代码预览、资料盘读取、部分反馈字段权限。
2. 马成海怀疑陈浩然，韩子昂提醒公共网络不能证明是谁。
3. 陆远要求先收权限，不能把怀疑当结论喊出去。
4. 团队冻结临时账号、收紧代码仓库和资料盘权限，统一对外说“防事故”。
5. 韩子昂发现旧测试账号仍有导出权限，重新跑权限表。

## 结果落点

访问日志、临时账号权限详情、导出按钮变灰、陈浩然私聊“你们这是防谁”。

## 连续压力

第308章马成海查到陈浩然私下接触竞品，第312章运营方案泄露时权限洞正式爆出来。
