# 第269章：三个流失用户等着他

- 所属卷：卷五：学伴杀出校园
- 故事阶段：投资人与条件
- 本章作用：用三类流失/沉默用户回访打断增长喜报，逼团队从真实痛点开始重排优先级
- 出场人物：陆远、林夏、马成海、韩子昂、刘鹏、顾明、南湖沉默用户、十分钟卸载用户

## 事件推进

1. 陆远暂停新增喜报，要求先听退过款、沉默高频、十分钟卸载三类用户原话。
2. 顾明骂“说得太满”，指出退款入口和处理人缺失。
3. 南湖沉默高频用户说自己怕麻烦，不敢付费；十分钟卸载用户指出开屏信息和活动弹窗劝退。
4. 团队把三类用户写成白板欠条：卡在哪里、谁处理、什么时候验。
5. 顾明约定明天中午来创业园大厅现场复测。

## 结果落点

三通录音、白板三列、顾明预约复测截图、被废掉的增长喜报。

## 连续压力

第270章顾明到创业园大厅复测，说出更刺耳真话。
