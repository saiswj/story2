# 第290章：林夏把留存摔给他看

- 所属卷：卷五：学伴杀出校园
- 故事阶段：团队内耗与产品重构
- 本章作用：用沉默用户流失打断团队报喜，让陆远更有血肉地面对怕输和想赢
- 出场人物：陆远、林夏、马成海、韩子昂、刘鹏、秦川

## 事件推进

1. 马成海展示新增和退款稳定数据，陆远也想把结果发给秦川。
2. 林夏点开沉默用户路径，证明部分用户不是因为价格离开，而是看不懂自己能不能用。
3. 陆远承认自己想躲开难看的曲线，最后关掉报喜对话框，先改待核验入口。
4. 新入口上线后，用户能看懂“能买、等核验、暂不适用”，秦川认可比报喜有用。

## 结果落点

沉默用户路径、待核验三色入口、用户“这回看懂了”的回访。

## 连续压力

待核验入口修了，但旧入口兼容马上出问题，推动第291章韩子昂进行核心架构修复。
