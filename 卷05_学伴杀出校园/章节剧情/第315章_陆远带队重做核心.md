# 第315章：陆远带队重做核心

- 所属卷：卷五：学伴杀出校园
- 故事阶段：竞品围剿
- 本章作用：陆远不把韩子昂推出去当替罪羊，而是带队重做反馈入口、处理状态和资料适配三条核心链路
- 出场人物：陆远、林夏、韩子昂、马成海、刘鹏、顾明、南湖老师

## 事件推进

1. 陆远投出旧版反馈入口，让团队直观看见用户为何会气。
2. 团队把反馈入口、状态栏、客服首回、超时标红拆成具体任务。
3. 林夏阻止陆远把最难任务全压自己身上，要求分工而不是独吞压力。
4. 新版核心链路在凌晨跑通，顾明十点复测确认入口、状态和暂不适用课程购买边界。
5. 南湖老师同意两门课试点继续，林夏独自去谈下午三点合作桌。

## 结果落点

三行白板、新版页面草图、状态栏、顾明复测截图、南湖老师下午三点邀约。

## 连续压力

第316章林夏独自去谈关键学校合作，把新版核心链路带上合作桌。
