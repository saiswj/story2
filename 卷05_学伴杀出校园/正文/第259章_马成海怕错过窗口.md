# 第259章：马成海怕错过窗口

马成海第二天来得最早。

他一进项目教室，就把三张打印纸拍在白板上。

第一张，云习社校园合伙人招募进度。

第二张，各学校互助群管理员的回复截图。

第三张，是他自己整理的入口流失表。

韩子昂还没到。

林夏刚把水杯放下，看见那三张纸，眉头就抬了一下。

“你通宵了？”

马成海嗓子有点哑。

“没通宵，睡了两小时。”

陆远走进来时，马成海已经把红圈画满了半块白板。

“南湖师院，云习社已经接触两个大群群主。”

“城南职院，他们给校园合伙人的佣金是每单二十。”

“江北财大，有人问我们能不能也给推广费。”

他越说越快。

“远哥，我知道韩子昂说得对，系统要稳。林夏也说得对，不能乱承诺。但入口不是等我们准备好了才打开的。”

马成海指着流失表。

“昨天从比价咨询跳走的用户里，有一百六十多个点进了云习社。不是他们一定不回来了，是他们已经被别人先装进漏斗。”

林夏拿起那张表，看得很仔细。

她没有反驳。

因为马成海这次不是凭感觉喊。

每个截图后面都有时间、来源、用户类别。

有核心付费用户问云习社能不能补充资料。

有观望用户说先买九块九试试。

也有群管理员很现实地问：

有人评论：“你们不给推广费，我凭什么把入口放在最上面？”

陆远把最后一句圈起来。

“这是直接问的？”

“嗯。”马成海说，“我没回。”

“为什么现在停？”

“我怕我回一句凭良心，然后人家把我拉黑。”


马成海没笑。

他坐下来，手撑着桌面。

“我昨晚想了一晚上。我们不能做虚假承诺，也不能用佣金让群主乱卖。但能不能有一个正式的校园志愿者计划？”

林夏问：“志愿者还是推广员？”

“先不卖。”马成海立刻说，“只做三件事。收课程版本，维护适用范围公告，组织答疑预约。完成任务给体验权益和后续优惠，不按成交拿钱。”

他又把一条私信放出来。

系统提示：“我愿意帮你们收课件，但别让我背销售指标。上一个项目让我拉人头，最后同学退不了款，都来骂我。”

马成海指着这条，声音低了点。

“不是所有群主都想赚快钱。有人只是怕被坑，也怕坑别人。我们如果不给一个干净入口，他们就只能在云习社那种合伙人计划和什么都不做之间选。”

林夏看他一眼。

“你这是把之前被砍掉的方案改了一版。”

“对。”马成海承认得很快，“因为之前那版太急，这版我认。但如果连反馈志愿者都不先占住，云习社会把人全招走。”

韩子昂这时进门。

他刚好听到最后一句。

“招走就招走。”

马成海火气一下上来。

“你能不能别每次都说得这么轻？”

韩子昂把电脑包放下。

“我不是说轻。我是说，不能为了抢人把权限放出去。”

“我说权限了吗？”

“你说维护适用范围公告。”韩子昂打开电脑，“谁维护？怎么审核？错了谁负责？”

马成海咬牙。

“那你给流程。”

韩子昂没想到他会这么回。

马成海把笔递过去。

“你别光说不行。你给一个能让人先进来的流程。”

项目教室安静了两秒。

陆远没有插手。

韩子昂接过笔，在白板上写：

手机亮起一行字：“反馈志愿者无后台权限。”

页面显示：“只提交课程版本和问题截图。”

公告写明：公告由团队审核后发布。

页面显示：“志愿者身份必须明示。”

页面显示：“禁止私下承诺折扣和效果。”

马成海盯着那几条。

“可以。”

林夏补了一句。

“再加退出机制。发现乱承诺，取消权益，公开更正。”


马成海立刻看向陆远。

“远哥，这个能做吧？”

陆远看完担心错过窗口的材料，点了点头。

“能做。”

马成海刚松口气。

陆远又说：“但不是今天全校推。”

“先拿三所学校试。南湖师院、城南职院、江北财大。每所不超过二十个反馈志愿者。”

马成海嘴唇动了动。

他想说太少。

但看见韩子昂和林夏都没反对，只能把话咽回去。

林夏说：“这是抢入口，不是抢成交。你要把这两件事分开。”

马成海低头看自己的表。

他当然知道。

可他更知道，周正阳以前每一次出手，都喜欢先抢位置。

现在他背后有孙启明。

位置一旦被抢走，再解释口碑，声音就会小很多。

上午十点，反馈志愿者报名页上线。

页面第一行不是“招募校园合伙人”。

而是：

页面显示：“帮助同学确认能不能用，不负责替学伴卖课。”

马成海看着这行字，表情有点复杂。

“一点都不热血。”

林夏说：“热血容易变味。”

“但很像你。”

“谢谢，少阴阳。”

报名页发到三个学校后，效果比预想中好。

半小时，南湖师院报名三十七人。

城南职院二十六人。

江北财大四十四人。

其中有几个人在备注里写得很直接。

那行字写着：“不想卖课，只想别再买错。”

页面上显示：“能不能先把老师课件版本核清楚。”

页面显示：“云习社找我了，但他们只问我能拉多少人。”

马成海看到最后一句，心里猛地一松。

“你们看，不是所有人都只看佣金。”

陆远说：“所以更不能把他们当销售。”

马成海点头。

就在这时，他的手机又响了。

是城南职院一个群管理员发来的截图。

截图里，周正阳正在他们校内一个大群里发消息。

私聊里写：“学伴太慢了。云习社今晚开放低价体验名额，先到先得。”

下面还跟了一句。

群消息刷出一句：“成熟团队，不拿学生当试验品。”

马成海的脸色瞬间变了。

他把手机递给陆远。

“窗口不是快关。”

他声音发紧。

“是有人在替我们关门。”

这一次，没人笑他急。

陆远把这次窗口期压力划到明早第一项。刘鹏重新排增长压力回访名单，先找那些不吵不闹、却已经停用的人。林夏说，增长压力里这类用户最容易被忽略，也最能说明问题。

沈老师的电话很快打进来。

她没有寒暄，第一句就问：“资料出错以后，谁替学生承担成本？”

马成海小声说：“我现在有点想念简单粗暴的骂战。”

林夏把免提打开，直接把问题推回去：“老师，我们今晚能补老师复核入口，但做不到每份资料都提前人工复核。”

电话那头静了两秒。

“做不到就写做不到。”沈老师说，“别让我明天替你们解释。”

陆远把公告最后一句删掉。

原本写着“我们将持续保障资料质量”。

删完后，只剩一句更难看的话：未经过老师复核的资料，将在页面标明来源和风险。

马成海看了半天，叹气：“这句发出去，用户会不会嫌我们怂？”

“嫌也比被骗强。”林夏说。

晚上十一点，沈老师回了一条语音：“按这个先走。明早我看入口。”

马成海看向陆远：“这算过关吗？”

陆远把电脑合上：“算她没关门。”

那一晚，公告里最难看的那行字留了下来。

窗口期没有被他们抢回来，却也没有被周正阳替他们关上。
