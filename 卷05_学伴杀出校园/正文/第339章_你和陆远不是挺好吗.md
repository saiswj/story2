# 第339章：你和陆远不是挺好吗

许晴以为自己已经把边界说清楚了。

她把公开邮箱发给二姨，也把陆远回的材料要求一条条转过去。学校资质、资料来源、退款规则、投诉处理人，缺一项都不能进合作池。

可亲戚真正想要的，从来不是邮箱。

晚上八点，家族群里忽然热闹起来。

二姨先发了一个笑脸。

二姨问：“晴晴，邮箱我们收到了。不过公开渠道肯定慢，你和陆远不是挺好吗？能不能帮忙提醒一下，让他们先看我们材料？”

这句一出来，群里跟着冒出几条。

有人跟着说：“是啊，认识的人说句话不一样。”

另一个亲戚补了一句：“又不是让他违规，就是打个招呼。”

还有人说：“现在陆远做大了，也不能连老熟人都不认吧。”

许晴看着“老熟人”三个字，指尖一点点发凉。

张芸坐在她旁边，脸色也不太好看。

许立明把眼镜摘下来，揉了揉眉心：“别回太冲。”

许晴轻轻笑了一下。

她其实没有想冲。

她只是突然发现，很多人根本听不懂“公开流程”这四个字。他们把流程当成门面，把关系当成真正的钥匙。

而她，曾经就是那把钥匙。

或者说，他们以为她还是。

张芸小声说：“要不我来说？”

“不用。”许晴拿起手机，“这件事该我说。”

她打了一行字，又删掉。

“我和陆远已经没有私人关系。”

太像撇清。

她又打。

“请不要再提以前。”

太像生气。

最后，她慢慢敲下去：

她在家族群里写得很直接：公开邮箱不是绕路，是唯一流程。陆远没有义务因为我优先看任何材料，我也不会替任何人打私人招呼。材料合规就按标准排，不合规就不要把学生当试错对象。

发送前，她停了很久。

张芸看着那段话，嘴唇动了动。

“晴晴，这样发出去，亲戚那边会觉得你不给面子。”

许晴没有抬头。

“以前我们给面子的时候，没给过陆远面子。”

张芸一下子沉默。

许立明叹了口气，把手机放到桌上：“发吧。”

许晴按下发送。

群里安静了十几秒。

然后二姨的语音打了过来。

许晴没有接。

语音挂断，文字又进来。

二姨私聊她：“晴晴，你怎么说得这么严重？我们也不是让你为难，就是觉得你们以前关系摆在那儿。”

以前关系摆在那儿。

许晴看着这几个字，心里像被轻轻刮了一下。

从前她觉得那段关系是她和陆远之间的事。

直到今天才发现，在很多旁观的人眼里，它还可以被拿出来估价、借用、讨人情。

张芸忽然站起来，拿走许晴的手机。

许晴一怔。

张芸没有替她软化语气。

她在群里发了一段话。

张芸写：“晴晴说得对。陆远现在做的是面向学生的项目，不是亲戚饭桌上的人情。你们要合作，就把材料按要求补齐。以后不要再拿两个孩子以前的关系说事。”

发完以后，张芸把手机还给许晴。

她的眼眶有点红。

“妈以前也说错过很多话。”张芸低声说，“这次不能再错。”

许晴看着她，一时没说出话。

家族群彻底安静了。

那种安静不是理解。

更像是大家忽然发现，这条旧路真的走不通了。

许立明倒了杯水，推到许晴面前。

“那陆远那边，要不要说一声？免得他们明天直接找过去。”

许晴摇头。

“不用让我夹在中间。”

她打开邮箱，把公开合作入口又发了一遍给二姨的私聊。

没有多写解释。

十分钟后，二姨终于回了。

二姨回：“行吧，那我们先整理材料。”

许晴把手机放下，胸口那口气才慢慢松开。

可事情没有就此结束。

晚上九点半，王雅婷发来截图。

论坛里已经有人把这件事传开了。

一条评论被顶上来：“听说许晴家亲戚找陆远项目合作，被要求走公开流程。”

下面评论很快歪了。

有人在评论区回：“前女友关系都不好使？”

另一条评论更直接：“陆远现在是真硬气。”

一条评论被顶上来：“许晴会不会尴尬死。”

许晴看着那些评论，没有像以前那样急着解释。

她只回王雅婷：

“不用管。”

王雅婷很快回：

“你真的没事？”

许晴看了一会儿屏幕。

她回：“有事，但能承受。”

这句发出去以后，她自己反而平静了些。

她不是不疼。

只是这一次，她没有再把疼推给陆远处理。

项目教室那边，马成海也看见了论坛帖。

他把截图发给陆远，问：

“要压吗？”

陆远回得很快。

“不讨论私事。合作材料按流程回复。”

马成海又问：

“有人带许晴节奏怎么办？”

陆远回：“不回应。”

林夏在群里补了一句：

“对外只说合作流程，不说旧关系。”

马成海看着这两条回复，第一次没有觉得不过瘾。

他把准备好的反击话术删掉，重新写：

团队号发出说明：“学伴所有合作方统一走公开邮箱，材料审核标准一致。涉及个人关系的讨论不回应。”

这条说明发出去后，论坛仍然吵。

但吵声没有再钻进流程里。

许晴看见那条说明时，已经快十点半。

她盯着“个人关系不回应”几个字，心里忽然一酸。

陆远没有羞辱她。

也没有保护她。

他只是把她从合作流程里拿了出去。

这比保护更难得。

因为那意味着他真的承认，她不该再被旧关系绑在他的项目上。

张芸从厨房出来，看见许晴还坐在沙发上。

“早点睡吧。”

许晴点点头。

她起身前，又看了一眼家族群。

没人再问“你和陆远不是挺好吗”。

那句话终于停下。

可许晴知道，明天还会有新的场面。

亲戚未必死心，论坛未必闭嘴，陆远那边也一定会把材料照流程退回。

这一次，她不会再站到流程前面。

临睡前，许晴又打开那封公开邮箱说明。

她把缺失材料三项抄到纸上，拍给张芸看。

“明天如果他们还问，就让他们补这个。”

张芸站在门口，轻轻点头。

母女俩谁都没有再提陆远以前来家里的样子。过去已经够拥挤了，不能再被拿去给别人铺路。

许晴关灯前，手机又亮了一下。

这回不是亲戚。

是王雅婷。

王雅婷发来一句：“你今天挺像个人。”

许晴看着这句不太好听的安慰，忽然笑了一下。

她回：

“那就从今天开始。”
