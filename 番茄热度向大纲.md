# 《都分手了，她表妹怎么追来了》大纲

## 题材定位

- 类型：男频都市校园、重生反舔狗、听劝系统、创业逆袭、前任后悔、表妹倒追。
- 核心爽点：前世舔狗惨败，重生后彻底不舔；别人随口一劝，男主真去执行，执行后立刻变强。
- 情感钩子：前女友以为男主离不开她，结果男主转身起飞；前女友的表妹本想替姐姐审判他，却比姐姐先动心。
- 连载口径：低门槛、高频爽点、强关系修罗场，每5章一个小钩子，每10章一次外部爽点，每30章一次公开爆点。

## 一句话卖点

前世我为校花女友耗尽青春，最后被她嫌弃没前途。重生大二开学，我绑定听劝系统，网友劝我别舔、搞钱、健身、拿奖，我全听了。可我没忘记前世那个堵住我的学妹：“你就是我姐那个前男友？”

## 简介

前世，大一暑假，许晴就说过他们不合适。

陆远没死心。分手之后，他仍旧给她送早餐，替她占座，帮她改论文，放弃竞赛和实习，只为换她一句“你真好”。

毕业那年，许晴彻底把话说死：“陆远，你给不了我想要的生活。”

再睁眼，陆远回到大二开学。

距离许晴提分手刚过去三天，他手里还拎着给前女友买的早餐。

前世记忆里，还有一个扎着马尾的学妹堵在实验楼门口，冷冷问他：“你就是我姐那个前男友？”

她叫林夏，是许晴的表妹。

这一次，他把早餐递给室友，转身绑定听劝系统。

网友劝他别舔，他不舔了。

网友劝他健身，他练出腹肌。

网友劝他搞钱，他做出校园爆款项目。

网友劝他离前任远点，他刚点头，许晴的表妹林夏就堵在实验楼门口。

“你就是我姐那个前男友？”

陆远：“前男友是真的，别的你自己看。”

后来，许晴红着眼问他还能不能重新开始。

陆远还没回答，林夏先站到他身边：“姐，他现在不归你管了。”

## 核心设定

### 听劝系统

陆远重生后绑定“听劝系统”。只要他收到外界建议并真正执行，就能获得奖励。

建议来源可以是论坛、短视频评论区、室友、老师、网友、用户、投资人，也可以是林夏的毒舌提醒。

系统不走慢热成长文，而是走番茄爽点闭环：

```text
建议出现 -> 系统评级 -> 陆远执行 -> 奖励到账 -> 外部震惊
```

### 系统数值

- 听劝值：执行有效建议后获得，可兑换技能、体质、学习效率和项目洞察。
- 建议评级：C、B、A、S。越难听、越真实、越能改变人生的建议，评级越高。
- 反舔值：拒绝无偿讨好、拒绝工具人任务、拒绝前任情绪勒索时获得。
- 爽点奖励：每次外部打脸成功，追加随机奖励。
- 失败惩罚：明知建议正确却继续舔，会扣听劝值、冻结奖励或短暂降低效率。

### 前20章硬闭环

前20章必须完成至少三次系统闭环：

1. 第3-5章：建议“别舔了”，陆远拒绝占座，获得反舔值，许晴第一次不适应。
2. 第7-10章：建议“别解释，拿成绩说话”，陆远小测反超，许晴主动靠近，林夏入场撞见。
3. 第13-20章：林夏毒舌建议“你这方案根本不是给人用的”，陆远改项目，Demo效果翻倍，林夏第一次动摇。

## 主要人物

### 陆远

男主。普通家庭出身，前世大一和许晴短暂恋爱，大一暑假被分手，却没死心，之后三年继续当免费工具人，错过竞赛、实习、创业窗口。重生回大二开学后绑定听劝系统，彻底断掉恋爱脑，开始学习、健身、赚钱、创业。

人物关键词：听劝、行动力、边界感、外冷内狠、越变越稳。

成长线：从“离不开前女友的普通男生”到“听得进真话、扛得住压力的年轻创业者”。

### 许晴

前女友。校花级学霸，家境稳定，现实、骄傲、理性。前世她嫌陆远没有规划、没有资源、没有未来。重生后，她发现陆远突然不再围着她转，并且越来越耀眼。

人物关键词：体面、骄傲、错判、后悔。

写作要求：不写成无脑恶女，但必须让她有明确错判和代价。她的后悔要有外部刺激：陆远拒绝她、林夏偏向陆远、周围人改口、项目公开获奖。

### 林夏

女主，许晴的表妹。大一新生，漂亮、聪明、嘴毒、行动力强。刚入学就知道陆远是表姐前任，一开始替表姐审判他，认定他是纠缠不清的旧人。后来发现陆远不解释、不卖惨、听得进难听话，还能把建议变成结果。

人物关键词：嘴毒、直球、护短、偏心、越看越上头。

关系线：误会审判者 -> 有效建议源 -> 公开护场者 -> 主动宣战者 -> 并肩创业者。

### 马成海

男主室友，嘴替和气氛担当。负责把读者想说的话说出来，也负责放大男主“不舔了”的爽感。

常用台词方向：“哥，你是真不舔了啊？”“以前你给她送早餐，现在她问你吃没吃饭，这世界终于正常了。”

### 周正阳

前女友身边的优质男追求者。家境好，会包装，会装深情。前期暗讽男主，中期抢署名、举报项目、抢资源，后期挖人、偷方案、投竞品。

人物功能：外部压迫源。每次出手必须有具体恶行和具体反杀，不许只会阴阳怪气。

### 韩子昂

男主创业合伙人。技术能力强，但社交弱，前期不相信陆远的商业判断，后来被男主执行力和用户洞察折服。

人物功能：事业线承接，团队扩容，防止男女主二人转。

### 唐明远

学院老师/项目导师。务实，不喜欢空话。前期只给男主一个边缘机会，后期在关键项目上认可男主。

人物功能：权威验收，提供校园项目入口。

### 秦川

投资人。卷四末出现，卷五正式入局。眼光准、条件狠，既给机会也制造压力。

## 关系结构

```text
陆远  ← 前世恋爱失败 / 今生彻底不舔 →  许晴
   ↑                                      ↓
   |                                  后悔与失控
   |
林夏：替表姐审判 -> 被能力打动 -> 公开偏心 -> 主动追求

周正阳：追求许晴，压制陆远，后期成为事业对手
马成海：室友嘴替，负责喜剧和爽点放大
韩子昂：创业合伙人，承接事业升级
唐明远：导师，负责项目验收和权威认可
秦川：投资人，负责资本压力和事业升级
```

## 主线结构

1. 重生醒悟：陆远从送早餐当天醒来，第一次不舔。
2. 表妹入场：林夏第1章以回忆亮相，第8章正叙登场，用“你就是我姐那个前男友”打出书名钩子。
3. 听劝变强：学习、健身、形象、竞赛、项目全面提升。
4. 前任失控：许晴第10章不适应，第20章吃醋，第30章第一次低头，第80章求复合。
5. 表妹偏心：林夏第20章动摇，第30章护场，第60章明显偏心，第80章主动宣战。
6. 校园创业：男主做出第一个校园爆款产品，获得老师、用户和投资人注意。
7. 修罗场爆发：前女友想回头，表妹公开告白，陆远开始默认她靠近。
8. 事业起飞：男主团队从校园项目走向市场，竞品和资本压力出现。
9. 终局合流：男主守住事业和感情边界，与林夏并肩登顶。

## 第二阶段番茄追读口径

第二阶段不能写成创业管理文，要写成升级后的强追读公司战争。

核心卖点：

- 真实输赢：陆远不是一直正确，卷11必须输到入口暂停、用户质疑、团队互怨。
- 高层对手：从周正阳这种校园反派，升级到沈亦舟、顾清禾、梁知行、程立言代表的竞合、资源、资本和规则压迫。
- 稳定CP压力测试：不写误会分手，不写男主情感动摇；写林夏在业务线上赢，陆远在边界上稳。
- 伙伴去留：有人离开，有人转岗，有新人加入，读者要感到公司长大的痛。
- 公司化副手：方知然负责把陆远的选择落成责任矩阵和组织动作，她能怼陆远但不抢主角；林夏绝不能被写成副手，她是恋人、合伙人和独立业务负责人。
- 条线负责人：韩子昂、唐若宁、高明川、刘鹏、马成海各守一条线，形成公司战支撑体系，不能写成男主一排小弟。
- 双层阴谋：卷11拆明线脏局，卷12短回甘后爆规则清洗局。

## 连载避坑

- 男主不能嘴上说不舔，行动还围着许晴转。
- 林夏必须在第1章以回忆方式亮相，第8章正叙登场，不能让书名卖点后置。
- 系统奖励必须外显，前20章至少三次完整闭环。
- 周正阳不能只会阴阳怪气，必须抢署名、举报、挖人、偷方案。
- 每次被污蔑后，必须有道歉、社死、资源反转、热榜扩散中的至少一个。
- 许晴不能太早深情，但必须快速失控：第10章不适应，第20章吃醋，第30章低头，第80章求复合。
- 林夏不能一上来倒贴，但必须快速偏心：第8章审判，第20章动摇，第30章护场，第60章明显偏心，第80章主动宣战。
- 事业线必须有可视化结果：榜单、用户数、付费、奖项、投资报价、竞品翻车。
- 系统惩罚只写扣听劝值、奖励冻结、效率下降，不写玄学霉运，避免降智。
- 首章前800字只保留“刚分手三天还买早餐、前世毕业被否定、林夏前男友钩子、早餐给室友、系统激活”五件事，其他舔狗细节分散到第2-5章补，避免首章解释过载。
- 卷五涉及团队内耗时，每5章必须挂一个外显结果：热榜、付费、入口丢失、测试账号抢回、竞品翻车、投资人报价、许晴亲戚态度反转，不能连续写内部会议。
- 卷五中段“方案成立”必须通过外部人承认：顾明愿意继续测、城北职院保留入口、南湖给测试账号、用户替团队反问竞品，内部表格不算兑现。
- 卷五中段必须让竞品短暂真正得手，再用公开数据反杀；只写复盘、白板、表格，不算爽点兑现。
- 108万轻补版不许为了过百万硬水。新增人物必须服务外部压迫、公开见证、用户反馈、合作卡点或谈判条款，不得新增纯日常朋友、纯吐槽同学或一次性工具人；陈露、邵宁、乔安、廖凡不进正式名单，竞品产品压力并入孙启明，避免新人稀释主线热度。
- 正文轻补已按该原则落地：正文源文件非空白字符约105.10万，根目录合订版含标题非空白字符约105.97万，重点增补卷05商业战和卷06收购谈判的可视化压力，每处新增场面均围绕谁急了、谁亏了、谁站队了、谁让步了、陆远/林夏获得了什么展开。
- 第二阶段每5-8章必须有小输赢或尾钩，每20章必须有公开结果，不能连续写会议、复盘、白板和制度讨论。
- 顾清禾线只写职业收编、路线竞争和产品定义权，不写深夜情绪依赖、私人暧昧饭局或情感替代。
- 卷12回甘控制在8-12章，甜点必须绑定入口恢复、团队分红、职位重排或审计异样，不写连续纯甜。
