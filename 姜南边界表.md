# 姜南边界表

> 姜南是优秀女性对手，不是感情第三者。她的功能是资源效率、产品定义权和路线诱惑。

## 功能定位

| 维度 | 写法 |
|---|---|
| 对陆远 | 职业认可、能力认可、收编欲、路线竞争、证明欲 |
| 对林夏 | 产品定义权和内容责任线的专业压迫 |
| 对学伴 | 用大厂资源换入口、标准和推荐权 |
| 对沈亦舟 | 阶段合作、互相利用，不完全同盟 |
| 对周正阳 | 保持距离，最多把他当脏素材来源 |

## 可写与禁写

| 可写 | 禁写 |
|---|---|
| 姜南提出更快、更成熟、更商业化方案 | 姜南深夜安慰陆远 |
| 姜南试图让陆远接受大厂标准化路径 | 姜南和陆远私人饭局暧昧 |
| 陆远认可效率模型的业务价值，但当场立合作边界 | 陆远把职业认可误写成私人情感依赖 |
| 陆远第一次被试探就立边界 | 陆远隐瞒与姜南接触 |
| 林夏用业务结果赢 | 林夏低级吃醋、争宠 |
