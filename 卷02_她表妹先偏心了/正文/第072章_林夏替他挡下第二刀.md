# 第072章：林夏替他挡下第二刀

路演前半小时，第二波舆论又来了。

这次不是匿名角度图。

而是有人在论坛发了一篇长帖。

标题看似理性。

【学伴团队值得关注，但创始人私人关系是否会影响项目稳定？】

内容比昨晚那种脏水更难处理。

它不骂人。

也不直接造谣。

它把陆远、许晴、林夏的关系写成“潜在团队风险”，又把林夏的用户体验负责人身份写成“非正式成员深度参与”，最后问了一句：

【如果后续关系变化，学伴是否会失去关键用户体验能力？】

马成海看完，气得想把手机摔了。

“这不还是八卦吗？换个壳就叫风险分析？”

韩子昂皱眉：“这帖比昨晚那个更麻烦。它把问题包装成商业稳定性。”


陆远正要说话，林夏已经拿过手机。

“我来回。”

马成海愣住：“你回？”

林夏点开评论框。

陆远看向她：“不用把自己卷进去。”

林夏抬头。

“我已经在里面了。”

这句话说得很平静。

不是赌气。

也不是冲动。

她是真的知道，这个问题绕不开她。

如果她不说，别人就会继续把她写成陆远团队里的暧昧变量。

她不喜欢这个词。

变量。

她做了访谈，改了流程，站了现场，熬了夜，挨了骂。

这些不是暧昧变量。

是工作。

她在帖子下面回复。

【我是林夏。】

【第一，我参与学伴，是因为这个产品的问题真实，团队愿意听真实反馈。】

【第二，我负责的是用户访谈和现场体验流程，相关文档、修改记录、现场数据都可查。】

【第三，我不是“非正式关系变量”，我是承担了具体工作的人。以后如果不参与，也应当按交接流程处理，而不是按八卦想象处理。】

【最后，讨论项目请看分工、记录和结果。把团队成员的具体工作写成八卦，本身就是对项目的不尊重。】

她发完，项目教室里安静了。

马成海张了张嘴，半天才挤出一句。

“林负责人，牛。”

韩子昂点头：“这条回复很清楚。”

陆远看着林夏。

“谢谢。”

林夏把手机放下。

“别谢。我是在替我自己的工作说话。”

陆远点头。

“嗯。”

他明白。

所以更该谢。

林夏这条回复很快被顶上去。

评论区风向再次变化。

【这回复很硬。】

【把女性贡献写成私人关系附属，这句说得对。】

【林夏不是只会护短，她是真做了事。】

【项目风险可以问，但别把人家工作写成八卦。】

有人开始把她过去几天做过的事一条条列出来。

【现场快速登记入口是她提的吧？】

【课程反馈标签也是她改的。】

【付费意向里那个替代方案问题，也是她加的。】

【这叫非正式变量？这叫产品脑子。】

这些评论出现后，林夏反而不太自在。

她并不习惯被这样摆到台面上。

可她没有删回复。

因为她知道，如果自己退回去，别人就会继续把她的工作写成暧昧附属。

这一次，她必须站住。

当然，也有人阴阳怪气。

【这么急着出来说，是不是心虚？】

李小薇立刻在下面回复。

【她要是不说，你说她默认；她说了，你说她心虚。你是评论区永动机吗？】

这条回复也很快被顶上来。

林夏看到后，忍不住笑了一下。

“李小薇真闲。”

马成海说：“你室友战斗力也很强。”

“她只是爱吃瓜。”

“能吃能打，优秀。”

路演通知响起。

工作人员在门口喊：“学伴团队准备。”

陆远把电脑拿起。

林夏刚才那条回复，让团队气氛稳了下来。

不是因为舆论完全平息。

而是因为他们终于把话说清楚。

她不是谁的附属。

也不是谁的替代品。

她是承担了具体工作的人。

走向报告厅时，许晴站在走廊另一侧。

她显然也看见了林夏的回复。

林夏脚步停了一下。

许晴轻声说：“你说得对。”

林夏怔住。

许晴看着她：“你的工作不该被写成八卦。”

这句话来得很轻，却让林夏心里某个结慢慢松了一点。

她点头。

“谢谢。”

许晴又看向陆远。

“路演顺利。”

陆远说：“谢谢。”

没有多余的话。

但这一次，许晴没有再觉得自己被冷落。

因为她知道，现在不是她要情绪位置的时候。

路演开始前，林夏主动把用户体验部分拿了过去。

马成海小声问：“你确定？现在论坛还在吵。”

“所以更要我讲。”林夏说。

陆远没有替她做决定，只问：“你讲哪三件事？”

林夏把流程稿翻开。

“资料缺口怎么被用户提交，组队可信怎么用规则筛掉空话，反馈状态怎么让用户知道问题没有掉进黑洞。”

陆远点头：“就讲这个。”

她站上台时，台下有人明显在看她。

那些目光里有好奇，有八卦，也有审视。

林夏没有避开。

她打开第一页，不讲陆远，也不讲许晴。

只讲用户体验。

讲到最后，她放出一张现场流程图。

上面每个节点都标着负责人。

用户体验与现场流程：林夏。

她看向台下，声音很稳。

“如果有人质疑我为什么在这个团队，我的回答是，我负责这些。”

会议室安静了一瞬。

随后有老师点头。

陆远坐在台下，看着她把自己的位置讲清楚，眼神也跟着稳下来。

这一次，不是他替林夏挡舆论。

是林夏自己站出来，把舆论挡回去。

有老师提问：“你们怎么证明这些不是靠一时热度来的？”

林夏没有急着看陆远。

她自己翻到下一页。

“看三个数字。”

首日新增。

次日打开。

反馈处理后再次使用。

她把激光笔落在第三个数字上。

“如果只是热度，用户骂完就走。现在有一部分人骂完回来，是因为他们看见问题真的被处理。”

会议室里安静了几秒。

提问老师点头：“这个回答比热榜有价值。”

林夏握着激光笔的手终于松了一点。

她挡住的不是一句流言。

是把所有人重新拉回用户身上。

路演结束后，林夏那条回复仍然挂在热评第一。

有人把“不是非正式关系变量，是承担具体工作的人”截出来转发。

李小薇在宿舍群里发了一排鼓掌。

林夏看到时，有些不好意思。

她并不习惯把自己的委屈摆到台面上。

可这一次，她发现说出来并不可耻。

她做了事，就该让别人知道那是事，不是八卦。

陆远走下台时，对她说：“刚才那段，很重要。”

林夏别开眼。

“路演也还行。”

马成海在旁边哑着嗓子感叹：“你们夸人都这么克制，累不累？”

林夏没理他，低头把路演问答里的用户体验部分重新标了一遍。

她刚才在台上看得很清楚。

有老师听到“真实差评”时明显感兴趣，也有老师对“付费意向”皱了眉。

下一轮，他们还会被问得更细。

挡住舆论只是第一步。

真正站稳，还要靠更硬的答案。

路演结束后，有个老师单独叫住她。

“你刚才那页流程图，明天能发我一份吗？”

林夏愣了一下：“可以。”

老师点头：“你们团队现在最容易被外界讨论关系，但你刚才讲清楚了岗位。年轻团队最怕贡献含糊，你们这点要继续保持。”

林夏拿着材料回到队伍时，马成海立刻凑过来。

“老师夸你了？”

林夏把材料拍到他怀里：“老师让你明天别漏数据。”

马成海一脸不信。

陆远看了她一眼，没拆穿。

林夏也没解释。

她只是忽然觉得，原来把自己的位置说清楚后，真的会有人按位置看她。

回到项目教室后，陆远把老师提到的问题一条条记进复盘表。

付费意向怎么验证。

团队分工怎么稳定。

课程反馈边界怎么持续执行。

林夏看着这些问题，心里反而没有刚才那么乱了。

被问，说明他们真的站到了下一层台阶上。

而下一层台阶，不会再允许他们只靠热度往前冲。

她把自己的流程图也发进群里。

【明天按这个补材料。】

马成海回得很快。

【收到，林负责人。】

这一次，林夏没有纠正他。

她把流程图保存成PDF，又给文件名加上日期。

这个动作很小。

可对她来说，像是把今天站上台的那几分钟认真收好。

不是为了证明她和陆远有什么。

而是为了证明她自己真的做过什么。

这两件事，终于被她分清楚了。
