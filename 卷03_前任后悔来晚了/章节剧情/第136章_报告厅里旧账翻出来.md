# 第136章：报告厅里旧账翻出来

- 所属卷：卷三：前任后悔来晚了
- 故事阶段：林夏公开告白与陆远护场
- 本章作用：在报告厅公开问答中引爆周正阳的身份攻击，逼林夏把八卦拆回团队稳定性问题
- 出场人物：陆远、林夏、许晴、周正阳、马成海、韩子昂、孙启明、主持人

## 事件推进

1. 学伴与云习社同组路演，林夏感到现场目光都盯着“许晴表妹”身份。
2. 周正阳在云习社问答中把私人关系包装成团队稳定性风险。
3. 林夏拦住陆远，反问周正阳到底是在问团队稳定性，还是替许晴审判她。
4. 林夏拿出访谈记录、脚本修改、社团试点反馈，要求问题回到岗位和贡献。

## 结果落点

报告厅公开发难、林夏通道边反问、资料夹第一页贡献记录、许晴后排脸色发白。

## 连续压力

许晴意识到自己不能继续沉默，第137章公开拆掉周正阳借她发难的道德牌。
