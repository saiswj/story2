# 第143章：训练营的门打开

第二天上午，训练营名单公布。

学伴进入下一轮。

云习社也进了。

这个结果让马成海当场骂了一句。

“他们都翻车了还进？”

韩子昂说：“他们数据也不差。补贴用户规模确实高。”

林夏看着名单。

“而且评审不会因为口碑翻车就直接淘汰。商业项目看的是能不能修。”

马成海叹气。

“你们能不能不要这么理性？我想爽一下。”

陆远把通知放到大屏。

“能进下一轮，就说明后面还会正面碰。”

唐明远很快来到项目教室。

他没有夸。

第一句话就是：“名单出来了，别以为稳了。”

马成海低声：“老师真是一点糖都不撒。”

唐明远看他。

“你想吃糖还是想过下一轮？”

马成海立刻坐正。

唐明远把下一轮要求写到白板上。

用户增长复盘。

商业模型验证。

团队风险处理。

外部竞争应对。

“这四项，下一轮都会被问。”

陆远点头。

“我们分工。”

韩子昂负责技术承载和反馈系统。

马成海负责社群增长和渠道质量。


林夏负责社团试点规模化和用户体验沉淀。

陆远负责整体商业模型和答辩。

唐明远听完，点了点头。

“林夏的部分要写细。”

林夏抬头。

唐明远说：“你现在不是临时帮忙的人了。你的工作如果沉淀不下来，团队风险就是真风险。”

林夏认真点头。

“我知道。”

这话让她心里很踏实。

唐明远不是因为她和陆远的关系提醒她。

而是因为她的岗位重要。

马成海把名单又看了一遍。

“云习社排在我们前面？”

陆远看过去。

下一轮展示顺序里，云习社仍然在学伴前一位。

周正阳显然也看见了。

他在论坛发了一条动态。

【下一轮见。】

马成海冷笑。

“还装。”

林夏说：“别回。”

马成海举手：“我现在已经成熟了。”

三秒后，他又小声问：“那我能不能用小号看？”

韩子昂说：“成熟得不多。”

项目教室里笑了一阵。

笑完，陆远把下一轮时间圈出来。

“三天后。”

林夏算了一下任务量，吸了一口气。

“今晚又要熬。”

陆远看向她。

“你先休息。”

林夏挑眉。

“你现在是以负责人身份说，还是以被公开追的人身份说？”

马成海差点把水喷出来。

陆远停了一秒。

“以希望主讲别倒下的负责人身份。”

林夏满意点头。

“勉强接受。”

训练营下一轮正式开始前，学伴所有人都知道，真正的压力还在后面。

秦川的问题，也会在下一轮出现。
训练营下一轮通知发来后，团队第一件事不是上课。

是填一张更细的团队风险表。

马成海看见表头，整个人都蔫了。

“怎么又是风险？”

唐明远站在旁边。

“因为你们现在最不缺热度，缺的是经得起问。”

表里第一项就是核心成员分工。

陆远写产品方向。

韩子昂写技术链路。

林夏写用户体验和社团试点。

马成海写社群反馈和对外传播。


写到“替代方案”时，林夏停住。

如果她临时不能参与，用户体验怎么继续？

这个问题比外面的骂声更扎。

因为它不是攻击。

它是事实。

陆远把她的用户体验三问放到模板库。

“先把你做过的东西沉下来。”

韩子昂补了一个字段。

【每次访谈必须记录原话，不只写总结。】

马成海补充：

【社群反馈分类，不许只按吵不吵。】

林夏看着大家一条条往她的工作里补东西，忽然有点想笑。

她曾经怕自己的位置被“流程化”后就不特别。

现在她发现，能被大家一起接住的位置，才不会轻易被舆论推倒。

训练营第一天结束，唐明远收走表格。

“下一轮，秦川会看这个。”

马成海哀嚎。

林夏却把笔收好。

“看就看。”

训练营的第一张风险表交上去后，很快被打回来。

批注写得毫不客气。

【风险识别充分，解决方案偏空。】

马成海看见，差点吐血。

“我们写了一晚上，他说偏空？”

唐明远把表拍到桌上。

“他说得对。”

陆远没有反驳。

他们把每一项风险后面都补上触发条件和处理人。

资料审核错误：刘鹏初筛，韩子昂系统标记，陆远终审，林夏负责用户通知话术。

社群舆论跑偏：马成海先分类，恶意攻击留档，产品问题转反馈池。

林夏个人争议：团队分工页对外公示，不由林夏单独解释。

写到最后一项时，林夏心里一动。

她发现自己不再是风险表里一个孤零零的名字。

她是流程里的一个节点。

也有别人和她一起承担。

第二版交上去后，导师只回了两个字。

【可用。】

马成海盯着那两个字，比看见夸奖还激动。

“可用！我们从偏空变可用了！”

林夏笑了一下。

创业训练营第一天，她学会的是把漂亮话压成责任。

训练营通知发来时，群里还附着一份导师名单。

马成海一眼看到秦川的名字，声音都变了。

“他真来？”

陆远把名单放大。

秦川后面标着：观察嘉宾。

林夏看着那四个字，手里的笔停了一下。

唐明远说：“别把观察嘉宾当贵人，也别当敌人。他问你们，是因为你们有问题值得问。”

这句提醒让教室里的兴奋慢慢沉下来。

被看见是机会，也是压力。

唐明远让他们把第二版风险表打印出来。

“每个人签名。”

马成海吓了一跳：“这么正式？”

唐明远说：“签了名，说明你知道自己负责什么。”


马成海看见自己的名字后，认真了几分。

“我要是真出错呢？”

陆远说：“先报，再补，别藏。”

这四个字，比任何口号都实在。

林夏在用户体验那一栏签下名字。

笔尖落下时，她有种很清晰的感觉。

她不是因为陆远才被写进这张表。

她是因为自己做过的事，被这张表承认。

签完后，陆远也在产品方向那一栏写下名字。

两个人的名字隔着几行，却在同一张责任表里。

马成海还想拍照留念，被唐明远一句“先做事”按了回去。
